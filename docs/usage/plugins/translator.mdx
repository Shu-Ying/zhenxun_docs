import Messenger from "/src/components/Messenger"

### 说明

通过配置百度翻译API，实现让小真寻帮你翻译。

- 插件类型: `NORMAL`
- 响应规则: `全局`
- 插件等级: `5`

### 指令

- 翻译语种
  - 说明: 查看翻译语种的执行代码

- 翻译 [文本内容] ?[-s zh] ?[-t en]
  - 说明: 翻译文本内容，-s为来源语言，-t是翻译语言。当任意参数被忽略，则默认为auto

### 示例

<Messenger
  msgs={[
    { position: "right", msg: "翻译 黑色 -s zh -t jp" },
    {
      position: "left",
      msg: `ブラック`,
    },
    { position: "right", msg: "翻译 猪 -t en"},
    {
      position: "left",
      msg: `pig`,
    },
  ]}
/>
