import Messenger from "/src/components/Messenger"

### 说明

每当新成员入群时，会发送欢迎消息。

- 插件类型: `NORMAL`
- 响应规则: `全局`
- 插件等级: `5`

### 指令

- 设置欢迎消息 [欢迎文本] ?[-at]
  - 说明: 设置群欢迎消息，当消息中包含 -at 时会at入群用户
